# 🔐 Security

## Security Officer
**Sebastian “ZeroDay” Thornfield**

**Role Summary:**
Owns company-wide information security, data protection practices, and secure development lifecycle alignment.

**Responsibilities:**
- Define and maintain security policies and standards
- Oversee application security (AppSec) reviews
- Conduct vulnerability assessments and coordinate remediation
- Manage access controls and identity governance
- Monitor cloud security posture
- Lead incident response and breach procedures
- Support compliance alignment (e.g., GDPR, SOC2 readiness)

> In startup mode, the Security Officer works closely with the CTO and DevOps Engineer and may contribute to secure coding reviews.
